# Feature Spec: Campus "View on Map" — Punla (UPLB Freshman Planner) This document describes an existing feature in `index.html` (a single-file vanilla JS PWA) so another AI assistant can understand, maintain, debug, or extend it without re-deriving the design from scratch. ## 1. What this feature does A **Campus** tab lists UPLB buildings as searchable cards. Each card has a **"View on map"** link that deep-links out to OpenStreetMap, centered and zoomed on that building's coordinates, opened in a new browser tab. This is a link-out, not an embedded map — no map library (Leaflet, Mapbox, etc.) is used, and no internet is required to browse/search the building list itself, only to actually view the map. ## 2.
